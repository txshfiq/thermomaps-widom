# Widom-Rowlinson Model using Thermomaps

## Description
This repository shows the usage of Thermodynamic Maps (TMs) to predict phase transitions in the multicomponent Widom-Rowlinson model by only sampling either side of the phase transition. This project was mostly inspired by the following paper:

> Herron *et al.*, "Inferring phase transitions and critical exponents from limited observations with Thermodynamic Maps," 	*PNAS*, **121**, 52, 2024. (https://doi.org/10.1073/pnas.2321971121)

Most of the work surrounding the Widom-Rowlinson model in this repository was inspired by the following paper:

> Lebowitz *et al.*, "Ordering and demixing transitions in multicomponent Widom-Rowlinson models," *Phys. Rev. E*, **52**(6), 5985–5996, 1995. (https://doi.org/10.1103/PhysRevE.52.5985)
